# Marketplace Frontend

This is the frontend application for the Marketplace platform, built with React, Vite, and Tailwind CSS.

## Features
- Real-time updates
- Merchant and Customer flows
- Order tracking and Finite State Machine (FSM)
- Dual language support (Arabic/English)

## Run Locally

**Prerequisites:** Node.js 20+

1. Install dependencies:
   `npm install`
2. Create your `.env` file based on the deployment guide.
3. Run the app:
   `npm run dev`

## Deployment
This application is configured to be deployed on Railway using the provided `railway.json` file.
